What Are WireFrames?

Wireframes are like the basic outline of your website. It's a design of how your website is basically gonna look like. We use wireframes because it helps get a basic idea of how our website is gonna look like. Its a template that helps us know where gonna put things and it helps not get stuck on designing our website.

Git Workflow

When collabrating with a group on a project each indivdual would have to make their own branch. To make you branch you would use "git checkout "name"". You could also use this command to change branches. There will always be a master branch in which contains the final product of your project and which other branches should merge their work to. After making you branch and working on somethings you would push it up towards your branch by using "git push -origin "name" ". After pushing to your branch you would go to the GitHub website and make a pull request so that you can merge your work into the master branch
